Configuración inicial:

Iniciamos un nuevo repositorio de Git en un directorio local

        
          git init

Configuración de nombre y correo electronico: 

        
          git config --global user.name "Nombre Apellido"
          git config --global user.email "email@ejemplo.com"

Clona (copia) un repositorio remoto en tu máquina local.: 

        
          git clone https://github.com/usuario/repo.git

Estado y seguimiento:

Con "git status" podemos ver los cambios que se han hecho en los archivos de nuestro repositorio.

          
            git status

Con "git add" podemos añadir archivos cambiado.

  • "git add nombreArchivo.txt" para actualizar archivos especificos.
  • "git add ." para actualizar todos los archivos modificados

    •           
            git add nombreArchivo.txt 

            git add .

    Con "git push" podemos subir los cambios al repositorio remoto, en mi caso uso github.

              
            git commit -m "actualizando mi código"
